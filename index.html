What's Inside

Seven tools, one workflow

Everything on this site is free, no login required. Here's what's available and what each tool does.

SEP-1 Tool

SEP-1 Time Zero

Enter clinical events and the tool auto-detects SIRS, organ dysfunction, and bundle elements — then suggests Time Zero (SSPT) with step-by-step reasoning.

Open tool LKW Tool

LKW Resolution Tool

Enter chart findings and resolve Last Known Well date, time, and Y/N values using TJC v2026B priority rules. Includes a practice quiz and case walkthroughs.

Open tool CMO Tool

CMO Abstraction Tool

Classify Comfort Measures Only denominator exclusions. Navigate POLST evaluation, timing rules, and common pitfalls — with a quiz and case walkthroughs.

Open tool HBIPS Tool

HBIPS Abstraction Tool

Calculate psychiatric inpatient hours, restraint and seclusion abstraction, age strata, and rates for HBIPS-2 and HBIPS-3. Includes a quiz and case walkthroughs.

Open tool Q&A Lookup

Q&A Lookup

Searchable bank of abstraction scenarios with spec-grounded answers. Filter by measure set (Stroke, ED, HBIPS, SEP-1), difficulty, or keyword tag. Export as PDF.

Open tool Tag System

Tag System

Built into the Q&A Tool — entries tagged with chart language abstractors actually use. Browse by category, filter by tag, or suggest new ones for review.
