How It Works

Search the way you actually think

No measure codes required. Type what you see in the chart and get a plain-language answer.

Search

Type what you see

Use chart language, abbreviations, or plain description. LKW, "disposition," "eloped," TNK, comfort measures. The tool finds it.

Filter

Browse by keyword tag

Every entry is tagged with the exact terms abstractors use. Click any tag to filter instantly.

Read

Get a plain-language answer

Guidance with the spec rationale behind it. Know not just what to abstract, but why, so you can explain your decision.
