ℹ How to use this tool ▾

Enter the charted time and a brief description for each clinical event relevant to SEP-1 review. The tool automatically identifies event types (SIRS, organ dysfunction, bundle elements, vasopressors) and builds a chronological timeline. Replace the example entries below with your patient's documented events. Include all findings that may affect Time Zero determination.

SIRS events: temperature, heart rate, respiratory rate, WBC, bands — e.g. Temp 38.9°C, HR 114, WBC 18.2
Organ dysfunction: lactate, creatinine, AKI, AMS, MAP, platelets, bilirubin — e.g. Lactate 3.8, Cr 2.1 (baseline 0.9)
Bundle elements: blood cultures, antibiotics, fluids, repeat lactate — e.g. Blood cx × 2 drawn, Zosyn 3.375g IVPB started
Vasopressors / shock: norepinephrine, vasopressin, dopamine, septic shock — e.g. Levophed started at 0.05 mcg/kg/min

Click any colored badge to manually override the detected event type. Leave Time Zero blank to auto-suggest based on SIRS + organ dysfunction pairing.
